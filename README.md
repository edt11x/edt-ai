# edt-ai
* You need to "npm install @openrouter/sdk"
* You need to set your Huggingface token in your environment, HF_TOKEN
* You need to set you Openrouter API Key in your environment, OPENROUTER_API_KEY
* You need to install a virtual environment for python3 so you can use pip
*   $ python3 -m venv myenv
* You need to install openai
*   $ pip instal openai
